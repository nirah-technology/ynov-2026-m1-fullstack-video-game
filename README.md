# Video Game

## Contexte

Découvrir l'architecture logicielle au travers du développement d'un jeu vidéo.

## Composants

- Profile du joueur
- Authentification
- Cartographie (régions du jeu + continents)
- Catalogue des entites (monstres), (stuff),  (race / classe / spells)
